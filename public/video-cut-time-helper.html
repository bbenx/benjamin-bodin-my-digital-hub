Ouvre cette page via le dev server : http://localhost:8080/video-cut-time-helper.html (port défini dans vite.config.ts).

Lance la lecture, puis pause pile à la fin voulue. Utilise ← / → pour reculer ou avancer d’une image (aperçu : bande-demo.mp4, ~60 img/s). Les commandes ffmpeg copiées ciblent toujours le fichier source IMG_3575.mov. Le temps s’affiche à la milliseconde.
